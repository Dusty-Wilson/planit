======
###A group trip organization tool.  
Calendars, trip map, activities schedule, expense tracking and splitting


  - Web based application built in Ruby on Rails
- Ruby on Rails
- Devise
- Google Maps API
- Google Calendar API
- PostgreSQL
- RSpec
- Deployed on Heroku
